Trevorchan


Installation

Requirements

  • PHP 4+
  • mySQL database

First Things First

Because you are viewing this file, you've made it as far as unzipping the file into a directory.
Now open up the directory inc. You should see a few files, including config.php. Open that up in your favorite text editor. (e.g. notepad)
It contains a bunch of default values being assigned to global variables. You need to edit them to your liking. For help, here is what each is for:

  • $dbconnection_host - The hostname of your mySQL database. Most people will use localhost, but it does differ.
  • $dbconnection_database - The name of the database which you will be using for the image board.
  • $dbconnection_user - The username you will be using to connect to the mySQL database.
  • $dbconnection_password - The password to the above username.
  • $chan_name - What you would like to name your image board site.
  • $chan_fancyname - The same above, but can be Formatted.
  • $chan_rootdir - This is the full internal path to the image board, you should probably leave this as the default.
  • $chan_webpath - The URL you will use to access your image board, such as http://imageboard.example.com. No trailing slash.
  • $chan_headerurl - The full URL to the header logo you would like to use. If you leave it blank, one will not be used.
  • $chan_maxppm - The number of posts any unique user may make per minute.
  • $chan_postdelay - The number of seconds any unique user must wait before making another post.
  • $chan_randomseed - Needs to be a lengthy string randomly entered, unique to your image board.

Now that your board has been configured, it's time to add the database. Included in the zip file should have been a file named trevorchan.sql, open it up and paste it into a SQL prompt, or upload it to the database using your favorite database manager. (e.g. phpMyAdmin)
If everything looks like it worked, upload the files (including the directories).

Getting Started

Add an Administrator Account

Now that your database is set up, your configuration is configured, and your files are uploaded, it's time to add a board. But first, we need to be able to log in to do so.
Open up your board in your browser. Visit the management panel, which can be done by accessing manage.php
If your board was at http://imageboard.example.com, you would visit the management panel by going to http://imageboard.example.com/manage.php.
You should see a very bare page, containing two input boxes, and a submit button. This is what it is supposed to look like.
This is where you will put your username and password when you want to log in. When the board is first created, there is only one user, an administrator with the username and password of "admin".
Log in with that combination, and you should see a variety of links to do different things. You'll learn about them all later.
For now, we need to secure your board. Click Staff.
Above the lists of administrators/moderators, there is a form to add a staff member. This is where you put in the username/password combination you desire. Fill them in, check the Is Administrator checkbox and click Add Staff Member.
Now click [logout]. This time log in with the combination you entered. If you successfully log in and see the panel just as you did before, then it worked. Log out again, and log in as admin.
Click Staff again, but this time delete the old admin account. Now only you can log into your management panel.

Add a Board

While logged in to the management panel, click Add Board.
You should see a short form. Think of a name for your first board, and the directory it should reside in.
If you can't think of one, try Random with the directory of b. In the directory box, put the directory for the image board, WITHOUT any /'s. In the description box, put the name of the image board.
Click the Add Board button. You should see a message telling you it worked.
But you're not done yet. Look for a file named board.htaccess. Copy this file into the directory of the new board on your host, and rename it to .htaccess. Congratulations, you just set up your first image board!

View and Post in the Board

Go ahead and visit your newly created image board. If you gave the board a directory of b, accessing it would be similar to http://imageboard.example.com/b/. Make a test post, which should get the ID of 1. Try replying to it and make sure the post ID's are properly incrementing.
If everything looks like it is working, move on to the next step.

Administrator Posting

Ruling your board with an iron fist may not be such a good idea, but it is nice to exercise some power once in a while.
Visit the management panel, and click on Get posting password. There should be a large, odd looking string of random characters. Copy the whole line to your clipboard. Now go back to your board. Fill in all the forms you were going to post with, but before clicking submit, open the password box. You do this by clicking the last e in Message. A box should appear. Paste your posting password into that box, and click submit. Once the board is finished regenerating, you should see ## Admin ## next to your name, in purple.
This works similarly for moderators, which can only do things to the board you allow them, and can not access the board administration pages. When they post, they get ## Mod ## next to their name, in red.
An administrator can also use a different password to appear as a moderator in their post (soon).
